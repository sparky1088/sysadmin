===== Sound =====
Sound not working correctly
{{{code: lang="sh" linenumbers="False"
killall pulseaudio; rm -r ~/.config/pulse/* ; rm -r ~/.pulse*
}}}

wait ten seconds, then run this: 
{{{code: lang="sh" linenumbers="False"
pulseaudio -k 
}}}

https://help.ubuntu.com/community/SoundTroubleshootingProcedure
