====== Vim ======

If you forget to sudo a file in vim you can always run:
{{{code: lang="sh" linenumbers="True"
:w ! sudo tee %
}}}


Sometimes pasting in vim has some weird format errors and paragraphs start to tab over.
This is fixable with
{{{code: lang="sh" linenumbers="True"
:set paste
i
}}}
